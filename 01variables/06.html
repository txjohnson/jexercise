Variables

A variable can be assigned the value of another variable. 

	let name1 = 'Clyde';
	let name2 = name1;
	name1 = 'Mary';

The variable name2 is given the current value of name1. Then name1 is given a new value. So by the end of execution, name1 is 'Mary' and name2 is 'Clyde'.
