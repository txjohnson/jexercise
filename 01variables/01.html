Variables 1.1

Variables... vary. They are a piece of memory that hold values. They're called variables because the values they hold can change. 

	var x;
	let y = 10;

The above introduces two variables. x is introduced but it has no value. It is undefined. y introduces a variable and set its value to 10. y is said to be initialized to 10.

Variables are created using var or let. The difference between the two is that var introduces a variable visible to the entire app. Meanwhile let introduces a variable visible only within the surrounding block (code in between { and }.
