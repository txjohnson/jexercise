Variables

A variable can be assigned a new value at any time using the '=' operator. 

	let color = 'red';
	color = 'green';

The variable 'color' starts out as 'red'. (It is initialized to red.) Then it is set to 'green'.

What value does x have after execution of the following code? 

	let x = 'Tic';
	x = 'Tac';
	x = 'Toe';
