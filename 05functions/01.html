Functions

A function is a block of code that can be called from other parts of a program. A function may or may not accept inputs and may or may not produce an ouput.

function speedLimit() {
    return 25;
}

This is an example of a function. This function accepts no inputs and produces a the number 25 as output. Calling this function, speedLimit(), will cause the string to be returned as the result

The () are important. They mark the inputs to a function. If a function has no inputs, then they are empty as shown. In Javascript, function declarations and calls need the () -- even when there are no inputs.

Note that the return statements is how values is returned from functions.
