Functions

A function is a block of code that can be called from other parts of a program. A function may or may not accept inputs and may or may not produce an ouput.

function double(n) {
    return n * 2;
}

This is an example of a function that takes a single input. Inputs to a function are called arguments or parameters. This function produces double what it is given as input. Calling this function, double(5), will produce 10

Remember, the () are important. They mark the inputs to a function. The inputs always go inside the (). If there are many parameters, they need to be separated by ,.
