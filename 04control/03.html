Control Structures 4.3

Control structures let you specify which block of code is executed or how many times execution will occur. 

let i = 55;
if (i > 64) { console.log ("too fast") }
else if (i < 45) { console .log ("too slow") }
else { console.log ("ok") }

The above only outputs "too fast" if i is more than 64, outputs "too slow" if i is less than 45 and otherwise outputs "ok". This is a called a general mutually exclusive condition. This type of condition has several parts: if that marks the start of a conditional statement, a test expression in () a block of code to execute in {} if the test turns out to be true, several else if fragments, and a possible else clause with a block in {} for when all else fails.

Logically, this is like a multi-way fork in the road. Only one path an be taken at a time. Each path is guarded by a test. If the test succeeds, then the path is taken. If no test succeeds, and there is a final else path, then that one is taken.
