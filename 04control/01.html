Control Structures 4.1

Control structures let you specify which block of code is executed or how many times execution will occur. 

let i = 55;
if (i > 64) { console.log ("too fast") }

The above only outputs "too fast" if i is more than 64. This is a called a conditional. A condition has, at least, three parts: if that marks the start of a conditional statement. A test expression in () and a block of code to execute in {} if the test turns out to be true.
