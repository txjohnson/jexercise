Control Structures 4.8

Control structures let you specify which block of code is executed or how many times execution will occur. 

let numbers = [5, 2, 8, 9];
let sum = 0;
for (let i = 0; i < numbers.length; i++) {
    sum += numbers [i]
}
console.log(sum)

A for loop has three parts: the for that declares what follows will be a loop, a head that sets the behavior of the loop inside the (), and the body, enclosed in {}, that will be executed 0 or more times.

The head has a number of parts as well. A semicolon (;) separates each section. The first part sets the initial state of the loop -- by declaring and initializing a variable or using a variable in scope. The second part, sets the termination condition. The final part specifies how the variable will change on each iteration.

We can add values to a list using push. 

let list = [];
list .push (5)

By the end of execution list has become [5].
