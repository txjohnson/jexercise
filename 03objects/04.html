Objects

Variables can be combined into a object. An object is like a dictionary of key/value pairs. These pairs are called properties. In object oriented programming, the key is called a member or member name of an object.

Like variables, the value of a member in an object can be changed using assignment (=) operator. 

let obj = { name: "Joe", age: 99 };
obj.age = 88;

The above declares an object and changes the age to 88.
