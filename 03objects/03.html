Objects

Variables can be combined into an object. An object is like a dictionary of key/value pairs. These pairs are called properties. In object oriented programming, the key is called a member or member name of an object.

Objects are created using {} notation. The value of each member can be a primitive Javascript type or another object.

let obj = { name: "Joe", age: 99 };

The above declares an object that has two properties: a name and an age.
