Arrays

Variables can be combined into a list. Each item in the list is a variable that refers to an an object or value.

Arrays are created using [] notation.

You can select an item in the array using object [] notation. 
let array = ['A', 'list', 'of', 'strings']
array [1]

The above declares a list that has four elements. The second line selects an element in the list by index number and yields 'list'.
